# Modelado de Datos SQL Cobranzas - Cobranzas y Analítica

## Descripción
Repositorio con ejemplos de modelado, normalización y consultas SQL aplicadas a procesos reales de cobranzas y recuperación de cartera.

### Objetivo
Demostrar habilidades en:
- Diseño de modelos relacionales
- Normalización de datos
- Construcción de indicadores de negocio
- SQL orientado a analítica y toma de decisiones
  
## Alcance 
El repositorio cubre:
- Modelado de entidades base (clientes, obligaciones y pagos)
- Relaciones entre entidades
- Consultas analíticas para indicadores de recuperación
- Enfoque en procesos de cobranzas

## Estructura del repositorio
- '01_clientes.sql': Información base de clientes
- '02_obligaciones.sql': Obligaciones financieras y estado de cartera
- '03_pagos.sql': Registro de pagos realizados
- '04_indicadores_recuperacion.sql': Métricas de recuperación y efectividad de pagos

## Contexto de negocio
Los ejemplos están basados en escenarios reales de cobranzas, integrando información de clientes, obligaciones y pagos para el seguimiento de indicadores clave de recuperación.
