# Modelado de Datos SQL Cobranzas - Cobranzas y Analítica

## Descripción
Repositorio con ejemplos de modelado, normalización de datos en SQL, orientados a procesos de cobranza y análisis de recuperación de cartera.
  
## Alcance del proyecto
Este proyecto representa un esquema relacional típico de una Compañía de financiamiento, permitiendo analizar información de clientes, obligaciones, pagos e indicadores de recuperación.

## Estructura del repositorio
- **01_clientes.sql**
  Creación de la tabla de clientes con información básica  y segmentación.
  
- **02_obligaciones.sql**
   Definición de las obligaciones financieras asociadas a cada cliente.
  
- **03_pagos.sql**
   Registro y control de pagos realizados sobre las obligaciones.
  
- **04_indicadores_recuperacion.sql**
   Consultas analíticas para medir recuperación de cartera  y comportamiento de pago.

## Tecnologías utilizadas
- SQL
- Modelado relacional
- Normalización de datos
- Consultas analíticas
